# Feedback Modal

## Назначение

Общий teacher-only паттерн для красивого показа успехов и ошибок вместо браузерных `alert()`.

## Где используется

- отправка Telegram-уведомления из `/students/[studentId]`;
- импорт студентов из XLSX на `/students`.

## UX-поведение

- открывается поверх текущей страницы с затемнением и blur-фоном;
- поддерживает состояния `success` и `error`;
- визуально опирается на текущую тему через токены `card`, `border`, `primary`, `destructive` и `status-success`;
- закрывается по клику на фон или по кнопке подтверждения.
