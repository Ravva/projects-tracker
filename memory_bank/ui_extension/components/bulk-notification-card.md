# Bulk Notification Card

## Назначение

Teacher-only карточка массовой Telegram-рассылки на странице `/students`.

## Поведение

- показывает список учеников с чекбоксами;
- позволяет выбрать всех или снять выделение;
- сразу показывает, у кого заполнен `telegram_chat_id`, а кто будет пропущен;
- отправляет одно сообщение выбранным ученикам через server action;
- выводит итоговую сводку по `sent`, `skippedNoChatId`, `skippedInvalidChatId` и списку ошибок через общий `FeedbackModal`.
