# App Shell

## Назначение

Базовый teacher-only каркас интерфейса для внутренних экранов продукта.

## Состав

- `SidebarProvider` и `TeacherSidebar`;
- sticky header с trigger и быстрыми действиями;
- `SidebarInset` как основная рабочая поверхность;
- глобальный `TooltipProvider` для интерактивных элементов shell.

## UX-паттерны

- плотная навигация без перегрузки цветом;
- фокус на статусах и weekly действиях;
- поддержка светлой и темной темы через единый набор токенов;
- возможность дальнейшего масштабирования на страницы `students`, `attendance`, `projects`.
