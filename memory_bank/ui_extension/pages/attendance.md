# Attendance Page

## Назначение

Teacher-only weekly workspace для контроля посещаемости.

## Маршрут

- `/attendance`

## Основные зоны

- weekly cards по трем плановым занятиям;
- таблица attendance grid по ученикам;
- действия `сохранить неделю`, `очистить неделю`, `удалить занятие`;
- заголовок weekday-колонки переключает весь урок по циклу, где четвертым состоянием идет `Не состоялось` с фуксией.
- при попытке уйти со страницы с несохраненными изменениями открывается модальное окно `Сохранить / Не сохранять / Отмена`.

## Потоки данных

Страница читает weekly lessons и student records через server-side repositories и пишет изменения через server actions. При открытии автоматически обеспечивается шаблон недели вторник/четверг/пятница; удаление занятия покрывает сценарий праздника. Статус `Не состоялось` хранится в `lessons.is_closed`, отключает редактирование ячеек этого дня и исключает урок из расчета weekly attendance rate.
