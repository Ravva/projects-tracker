# Product Context

## Для чего нужен проект

`Projects Tracker` нужен преподавателю, который ведет занятия по разработке приложений с ИИ и хочет в одном месте:

- отмечать посещаемость по неделям;
- видеть учеников в зоне риска;
- контролировать прогресс ученических GitHub-проектов;
- получать AI-сводки по проектам;
- отправлять уведомления через Telegram.

Дополнительно активирован ограниченный student-access сценарий: ученик после подтверждения через Telegram и входа через GitHub может выбрать свой репозиторий как проект, а преподаватель может подключить второго ученика к этому же проекту для работы в паре.

## Ключевые пользовательские проблемы

- посещаемость и проектный прогресс живут в разных инструментах;
- teacher тратит время на ручную привязку GitHub-аккаунтов учеников;
- `github_username` ненадежен как долгосрочный идентификатор;
- без централизованного AI-анализа сложно быстро увидеть проблемные проекты;
- Telegram-доставка требует подтвержденного `chat_id`, а не одного лишь username.
- при работе в паре один проект привязывается только к одному ученику, а второй выпадает из учета, если в модели нет общего project record и memberships.

## Зафиксированные продуктовые решения

- teacher control room остается главным сценарием продукта;
- student-access ограничен маршрутом выбора проекта и не открывает attendance, teacher dashboard или teacher CRUD;
- student identity строится на стабильном `github_user_id`, а не на `github_username`;
- Telegram deep-link используется как шаг подтверждения личности ученика перед GitHub bind flow;
- после `Start` бот должен отправлять ученику персональную GitHub login-ссылку;
- teacher-only экран редактирования ученика остается центром управления привязкой Telegram и служебными данными;
- student может выбрать только репозиторий из собственного GitHub repository list;
- один GitHub-репозиторий и один AI-analysis должны соответствовать одному project record, даже если над проектом работают два ученика;
- редактирование attendance, заметок, ручных override и teacher review остается только у преподавателя.
