# Active Context

## Текущий фокус
- **Интерфейс проектов по спецификации Untitled UI**: Идеальный редизайн страниц проектов (/projects), деталки, логов и отчетов завершен.
- **Унификация UI-стандартов**: Контейнеры переведены на rounded-lg (12px), интерактивные элементы — на rounded-md (8px), мелкие кнопки — на rounded-[6px] (6px). Убраны все размытые свечения у статусов в пользу плоских кольцевых обводок.

## Текущие решения
- Архитектурная основа в docs/README.md, визуальные правила — в DESIGN.md.
- Для шрифтов повсеместно применяется гарнитура Inter (font-sans) взамен кастомных системных стеков.
- Ветка разработки: main. Biome проверяет и автоформатирует измененные TS/TSX файлы.
