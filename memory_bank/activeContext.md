# Active Context

## Текущий фокус
- **Полный визуальный редизайн по спецификации DESIGN.md**: Начата поочерёдная переработка страниц проекта под дизайн-систему Untitled UI Classic (Slate).
- **Экран раздела учеников и детального профиля студента отполирован**: Полностью переработаны `/students`, `/students/[studentId]`, а также суб-компоненты `bulk-notification-card.tsx`, `import-button.tsx`, `github-link-card.tsx`, `telegram-link-card.tsx` и `notification-card.tsx` на базу токенов `DESIGN.md` (скругления 8px для кнопок/полей ввода, 12px для контейнеров/карточек, прозрачные glass-эффекты с размытием `backdrop-blur-md`, семантическое удаление инлайновых стилей).
- Ожидание запуска локального сервера разработки Хозяином (`bun run dev`) для совместной верификации интерфейса.

## Текущие решения
- Архитектурная правда зафиксирована в `docs/README.md`, визуальные правила и токены — в `DESIGN.md`.
- Интерактивные элементы строго ограничены скруглением 8px (`rounded-md` / `rounded-[8px]`), карточки — 12px (`rounded-lg`).
- Для текстовых контролов принудительно применяется гарнитура Inter (`font-sans`).
- Ветка по умолчанию: `main`. Для Biome настроены строгие правила `format` и `lint` (исключая markdown).
