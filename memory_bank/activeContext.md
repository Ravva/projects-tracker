# Active Context

## Текущий фокус
- **Интерфейс посещаемости по канонам Untitled UI**: Завершен идеальный редизайн страницы посещаемости (`/attendance`) и сводного отчета для родителей (`/attendance/report`).
- **Шлифовка кнопок раздела учеников**: На странице `/students` устранены устаревшие скругления `rounded-xl` и `rounded-lg` у кнопок действий («Импорт XLSX», «Добавить ученика», «Сохранить карточку», а также в списке строк). Кнопки приведены к единому стандарту Untitled UI: 8px (`rounded-md` / `size="sm"`) и 6px (`rounded-[6px]` / `size="xs"`).

## Текущие решения
- Архитектурная правда зафиксирована в `docs/README.md`, визуальные правила и токены — в `DESIGN.md`.
- Интерактивные элементы строго ограничены скруглением 8px (`rounded-md` / `rounded-[8px]`), карточки — 12px (`rounded-lg`).
- Для текстовых контролов принудительно применяется гарнитура Inter (`font-sans`).
- Ветка по умолчанию: `main`. Для Biome настроены строгие правила `format` и `lint` (исключая markdown).
