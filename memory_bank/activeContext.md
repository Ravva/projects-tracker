# Active Context

## Текущий фокус
- **Полный визуальный редизайн по спецификации DESIGN.md**: Рефакторинг раздела учеников, входа, личного кабинета, а также страницы посещаемости (`/attendance`) и отчетов родителей выполнен в контуре токенов Untitled UI Slate.
- **Консистентность интерфейса**: Удалены устаревшие скругления `rounded-xl`/`rounded-2xl` с кнопок и карточек, крупные размытые тени заменены на семантический контур, а шрифты выровнены по Inter/font-sans в соответствии с дизайн-системой.

## Текущие решения
- Архитектурная правда зафиксирована в `docs/README.md`, визуальные правила и токены — в `DESIGN.md`.
- Интерактивные элементы строго ограничены скруглением 8px (`rounded-md` / `rounded-[8px]`), карточки — 12px (`rounded-lg`).
- Для текстовых контролов принудительно применяется гарнитура Inter (`font-sans`).
- Ветка по умолчанию: `main`. Для Biome настроены строгие правила `format` и `lint` (исключая markdown).
